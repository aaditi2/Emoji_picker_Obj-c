# Emoji Picker (macOS • Objective-C)

A lightweight macOS menu bar app for quickly selecting and copying emojis using a clean AppKit-based interface.

---

## ✨ Features

- Menu bar app for instant access  
- Grid-based emoji picker  
- One-click copy to clipboard   
- Real-time emoji search (keyword-based; filters emojis as you type)
- Recently used emojis (auto-saved)  
- “Copied!” feedback toast 

---

## 🧠 How It Works

- `NSStatusItem` creates the menu bar app  
- `NSCollectionView` displays emojis in a grid  
- `NSUserDefaults` stores recent emojis  
- `NSPasteboard` handles copy functionality  
- Custom toast view provides visual feedback
